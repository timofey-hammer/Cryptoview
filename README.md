# Cryptoview
Технологии: SwiftUI, Combine, MVVM, CoreData.

Данное приложение позволяет осуществлять мониторинг курсов криптовалют и сохранять их в реальном времени.

## Содержание
- [Технологии](#технологии)
- [Использование](#использование)
- [Источники](#источники)

## Технологии
- Swift 5
- IOS 16.0
- Xcode 14.0
- SwitfUI
- Combine
- MVVM
- CoreData
